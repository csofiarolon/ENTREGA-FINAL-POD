NUBELANDIA

Seleccioná un juego:

Tormenta de rayos
Del cielo caen rayos. Mové la nube con las teclas de dirección (izquierda, derecha) para esquivarlos. Cada vez que un rayo no alcance la nube, tu puntuación subirá. Tenés 60 segundos para jugar. ¡Buena suerte!
Copos voladores
Entre las nubes caen copos de nieve. Usá las teclas de dirección (izquierda, derecha) para evitar que caigan al suelo, o perdés. Cuando un copo toca una nube, tus puntos aumentan. Tenés 60 segundos para jugar. ¡Buena suerte!
El deshielo
Dispará rayos láser con la nube para derretir los cubos de hielo en 60 segundos. Usá las teclas de dirección de izquierda y derecha para moverte y hacia arriba para disparar. Si los cubos llegan al suelo, perdés. Cada cubo suma puntos. ¡Buena suerte!
Cazador de gotas
La nube juega al ping-pong con gotas de agua. Atrapá las gotas para aumentar tu puntuación. Cada vez caerán más gotas. Si alguna gota toca el suelo, perdés. Tenés 60 segundos. ¡Buena suerte!
